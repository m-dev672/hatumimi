添付のPDFは大学の教育課程表です。このPDFから卒業に必要な単位数を以下の形式のJSONで出力してください：

{
  "中分類1": 単位数,
  "中分類2": 単位数,
  "中分類3,中分類4": 単位数,
  ...
}

【重要な条件】
1. 複数の中分類からOR条件で単位を取得できる場合は、カンマ区切りの文字列でキーを表現してください
   例: "展開科目,関連科目": 16
   
2. 必修単位は考慮しなくて大丈夫です

3. 合計は不要で、共通教養科目と専攻科目を同じJSON内にフラットに並べてください

4. 単位数の計算に注意してください
   例: 「展開科目及び関連科目から46単位以上。ただし、展開科目から30単位以上」
   → "展開科目": 30, "展開科目,関連科目": 16 (46-30=16)
